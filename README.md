# Bruno Bros

Jogo foi criado com inspiração no Super Mario Bros, porém esse aqui se chama Brono Bros, e assim que nasceu o Bruno, o personagem 2d feito para brilhar!

## Como jogar

Abra `index.html` no navegador.

- Mover: setas esquerda/direita ou `A`/`D`
- Pular: seta para cima, `W` ou espaço
- Celular/tablet: use os botões na tela
- Reiniciar: botão `Reiniciar`

Os gráficos e personagens são originais para evitar copiar assets protegidos.
